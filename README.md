# What is mmCNC?
mmCNC is a lightweight, easy to build desktop CNC machine designed for jobs like laser cutting/engraving and plotting. **This project is under development! I recommend not to download the models and start to build the machine yet!**
![Render of the machine](https://raw.githubusercontent.com/mosomate/mmcnc/refs/heads/main/docs/cover.png)
